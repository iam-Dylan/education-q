# EducationQ Dataset Files

Downloaded from the official EducationQ repository to match the paper setup.

Files:
- `gpqa/dataset/gpqa_diamond.csv`: GPQA Diamond subset, 198 questions.
- `gpqa/prompts/chain_of_thought_examples.json`: GPQA validation/few-shot support file used by the original config.
- `mmlu-pro/mmlu_pro_stratified.json`: MMLU-Pro Stratified subset, 1,300 questions.

Paper total: 1,498 questions across GPQA Diamond and MMLU-Pro Stratified.

Official source: https://github.com/SunriserFuture/EducationQ
